# 🗂️ My Tasks Board

![Node.js](https://img.shields.io/badge/Node.js-18+-green)
![Express.js](https://img.shields.io/badge/Express.js-Backend-black)
![EJS](https://img.shields.io/badge/EJS-Template-orange)

![License](https://img.shields.io/badge/license-MIT-green)

A simple **Task Management Web Application** built with **Node.js**, **Express.js**, **EJS**, **CSS**, and **Vanilla JavaScript**.

The application allows users to **create, view, edit, and manage tasks** through a clean and simple interface.

---

# 🚀 Features

* ➕ Add new tasks
* 📋 View all tasks
* ✏️ Edit existing tasks
* 🔍 View task details
* 🧩 Server-side rendering using **EJS**
* 🎨 Simple and responsive UI

---

# 🛠️ Technologies Used

* **Node.js**
* **Express.js**
* **EJS (Embedded JavaScript Templates)**
* **JavaScript**
* **CSS**

# ⚠️ Important Note

The **server file is located inside the `js` folder**.

When running the application, start the server using:

```bash
node js/server.js
```

---

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/My-Task-Board-Challenge.git
```

Navigate into the project directory:

```bash
cd My-Task-Board-Challenge
```

Install dependencies:

```bash
npm install
```

---

# ▶️ Run the Application

Start the server:

```bash
node js/server.js
```

Or using **nodemon**:

```bash
npx nodemon js/server.js
```

Open your browser and visit:

```
http://localhost:3000
```
